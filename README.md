Question 1: Difference Between Data Types
Integer (int)

An Integer is a data type used to store whole numbers without decimal points. It can be positive, negative, or zero.

Example: 10, 25, -8

Float (float)

A Float is a data type used to store numbers with decimal values.

Example: 3.14, 99.99, -0.75

String (str)

A String is used to store text or characters enclosed within single or double quotation marks.

Example: "Hello", "Python"

Boolean (bool)

A Boolean data type stores only two values: True or False. It is mainly used in decision-making and conditional statements.

Example: True, False

Question 4: Five Commonly Used String Methods in Python
1. upper()

Converts all characters of a string into uppercase letters.

Example: "python".upper() → PYTHON

2. lower()

Converts all characters of a string into lowercase letters.

Example: "PYTHON".lower() → python

3. replace()

Replaces a specified word or character with another word or character.

Example: "I like Java".replace("Java", "Python") → I like Python

4. strip()

Removes extra spaces from the beginning and end of a string.

Example: " Hello ".strip() → Hello

5. split()

Splits a string into a list based on a specified separator.

Example: "Apple,Banana,Mango".split(",") → ['Apple', 'Banana', 'Mango']

Question 7: What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is a branch of computer science that enables machines and computer systems to perform tasks that normally require human intelligence. These tasks include learning, reasoning, problem-solving, decision-making, understanding language, and recognizing patterns.

Importance of AI
Automates repetitive and time-consuming tasks.
Increases efficiency and productivity.
Reduces human errors and improves accuracy.
Helps in analyzing large amounts of data.
Enhances user experience through personalized services.
Four Real-Life Applications of AI
Healthcare and Medical Diagnosis
Self-Driving Cars
Virtual Assistants (Alexa, Siri)
Recommendation Systems (Netflix, YouTube)
Question 8: Identify Whether the Following Are Examples of AI
ChatGPT – AI

ChatGPT is an AI application because it uses machine learning and natural language processing to understand and generate human-like text responses.

Google Maps Route Prediction – AI

Google Maps uses AI to analyze traffic data, predict travel times, and suggest the fastest routes.

Calculator – Not AI

A calculator follows fixed mathematical rules and instructions. It does not learn from data or make intelligent decisions.

Netflix Recommendations – AI

Netflix uses AI algorithms to analyze user preferences and viewing history to recommend movies and TV shows.

Voice Assistants (Alexa/Siri) – AI

Voice assistants use speech recognition, natural language processing, and machine learning to understand voice commands and provide useful responses.
